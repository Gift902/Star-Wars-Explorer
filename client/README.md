PROJECT OVERVIEW
----------------
This is a built and small responsive website, built using React.js
that uses SWAPI to allow a user to browse a list of Star Wars characters
and view thier details about them.
I implimented the container and component pattern where container handles
data fetching and state management and components handles the display and 
rendering the data.

INSTRUCTIONS TO RUN IT LOCALLY
------------------------------
1. Open Visual Studio Code 
2. Create a folder in which you will store the project
3. Open the terminal (or use git bash if you have one)
4. And then type : git clone https://github.com/Gift902/Star-Wars-Explorer.git
5. And wait for the project to be cloned
6. After it is done open another terminal
7. And then type : cd client
8. And then type : npm run dev
